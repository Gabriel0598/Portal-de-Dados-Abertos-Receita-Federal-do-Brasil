# Portal-de-Dados-Abertos-Receita-Federal-do-Brasil
A receita federal disponibiliza alguns dados abertos sobre as empresas existentes hoje no Brasil. Esses dados são de domínio público e livre acesso, porém eles podem sofrer um desatualização de até 3 meses. Será realizada ingestão de tabelas a partir de um endpoint e realizar processamento de dados para geração de dados agregados.
